# Stock Price Predictor

Simple tools for analyzing company profiles and creating graphs used in a stock price prediction project.

Included files
- CompanyProfileAnalyzer.py
- GraphAnalyzer.py

Quick start
1. Create and activate a virtual environment:
```powershell
python -m venv .venv
.\\.venv\\Scripts\\Activate.ps1
```
2. Install dependencies (if any).
3. Run the scripts:
```powershell
python CompanyProfileAnalyzer.py
python GraphAnalyzer.py
```
